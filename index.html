Request Header Parser Microservice

By Devansh D. | Project GitHub

This website if for the Free Code Camp Back End API Request Header Parser Microservice project. It exposes a single enpoint at /api/whoami which returns infomation about your IP address, locale, and the software you are using in a JSON format.

User Stories:
  1. The endpoint can be found at: https://fcc-request-headers-parser-ms.herokuapp.com/api/whoami/api/whoami.

Example usage:

https://fcc-request-headers-parser-ms.herokuapp.com/api/whoami/

Example output:

{"ipaddress":"10.8.46.55","language":"en-US","software":"X11; Linux x86_64"}
